# AI-TOOL-FOR-EARLY-STAGE-DEMENTIA-DETECTION
An AI-powered web application designed to detect early signs of dementia using machine learning techniques. This system analyzes user inputs and provides predictions to assist in early diagnosis.
🚀 Features
🧠 Dementia prediction using Machine Learning
🌐 Interactive web interface
⚡ Real-time analysis and results
📊 Data preprocessing and feature extraction
💾 SQLite database integration
🛠️ Tech Stack
Backend:
Python
Flask
Machine Learning:
Scikit-learn
Pandas, NumPy
Frontend:
HTML, CSS
Tailwind CSS
JavaScript / TypeScript
Database:
SQLite (mindguard.db)
📂 Project Structure

⚙️ Installation
1. Clone the repository
git clone https://github.com/your-username/mindguard.git
cd mindguard
2. Install backend dependencies
pip install -r requirement.txt
3. Run the Flask app
python app.py
🌐 Frontend Setup
cd frontend
npm install
npm run dev
▶️ Usage
Start backend server
Run frontend
Enter user data
Get dementia prediction results
